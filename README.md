# East Asia Business — Financial System

Internal financial management system for East Asia Business Co., Ltd.

## Modules
- 📊 Dashboard (KPI, AR/AP, Charts)
- 💼 Financial Center (Sales Docs / Procurement / Expenses / Transactions)
- 📦 Inventory (Barcode, Pricing)
- 🌍 Customers & 🏭 Suppliers (Full Profiles)
- 📈 Reports & CSV Export

## Tech
- Single-file HTML + JavaScript
- localStorage (Phase 1)
- Supabase integration planned (Phase 2)
